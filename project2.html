Project description

A simple Whatsapp desktop inspired real-time chatting application. Supports embeds from popular social media sites (i.e. YouTube, imgur). Written with a MERN stack with a self-built chat engine and email verification using nodemailer and Gmail.

Operating 24/7 Live demo is not online due to Heroku disabling its free dynos
